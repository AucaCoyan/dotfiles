# `pipx` is a tool to install packages from PyPI into isolated enviroments

Install pipx [here](https://pypa.github.io/pipx/installation/)

and remember to run `ensurepath`

Here is the list:

- `all-repos`
- `black`
- `flake8`
- `mypy`
- `pipenv`
